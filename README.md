# Whirlpool Sales Dashboard

## Overview
Interactive sales dashboard built in Power BI analyzing Whirlpool's sales data.

## Key Metrics
- Total Revenue: $5.66 bn
- Total Units Sold: 177.52k
- Dealer Count: 517
- Avg. Selling Price: 31.87k

## Charts Included
- Revenue by Financial Year
- Revenue by Channel & Category
- Revenue by StateName
- Unit Sold by Product Category
- Unit Sold by Region Name

## Tools Used
- Power BI
- DAX
- Data Modeling
